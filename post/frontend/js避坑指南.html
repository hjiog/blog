# 排错过程

  1. 在开发者工具查看network时发现webworker/index.ts还额外引入了含有dom操作的test.ts
  2. 原因是vite在开发时没有做到treeshaking,import xxx from '/common'会将common的暴露的所有变量给导入进来，自然也将test.ts导入了
  3. 正确做法是将test.ts从common/index.ts移除即可
